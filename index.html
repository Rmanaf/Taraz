روش استفاده

کتابخانه "تراز" را به برگه اضافه کنید

<head>
    <script src="https://rmanaf.github.io/taraz/latest/jquery.taraz-digits.min.js"></script>
</head>

روش اول بدون استفاده از javascript

  • کلاس auto-fix را به تگ html اضافه کنید

  • به بخش موردنظر کلاس fix-digits را اضافه کنید

  • پس از بارگذاری برگه اعداد داخل متن پارسی می شوند

<html lang="fa_IR" class="auto-fix">
    ...
    <span class="fix-digits" >10000</span>
    ...
</html>

برای فیلد ها

و در متن ها 1200456771299

روش دوم با استفاده از متد تراز

$.fn.taraz(["digits"])

و یا با انتخاب تگ بدون نیاز به کلاس fix-digits

$("<selector>").taraz(["digits"]);

روش استفاده

کتابخانه ها را به برگه اضافه کنید

<head>
    <script src="https://rmanaf.github.io/taraz/latest/jquery.taraz-currency.min.js"></script>
    <script src="https://rmanaf.github.io/taraz/latest/jquery.taraz-digits.min.js"></script>
</head>

روش اول بدون استفاده از javascript

  • کلاس auto-fix را به تگ html اضافه کنید

  • به بخش موردنظر کلاس های fix-currency و fix-digits را اضافه کنید

<html lang="fa_IR" class="auto-fix">
    ...
    <span class="fix-currency fix-digits" >10000</span>
    ...
</html>

برای فیلد ها

و در متن ها 1200456771299

روش دوم با استفاده از متد تراز

$.fn.taraz(["currency" ,"digits"])

و یا با انتخاب تگ بدون نیاز به کلاس های fix-digits و fix-currency

$("<selector>").taraz(["currency" ,"digits"]);
